# Amazon One at NYUL Hospitals

## Overview
Developed a secure, contactless identification system for hospital check-ins. Focused on **high reliability, data security, and efficiency**.

## Features
- Real-time biometric scanning for patients and staff
- Secure backend authentication
- High concurrency handling
- GDPR-compliant data handling

## Tech Stack
- Python, Flask, PostgreSQL
- Docker, Kubernetes

## Highlights
- Reduced check-in wait times by 40%
- Contributed to national interest by providing **secure identification processes**
- Implemented logging and monitoring for operational reliability
